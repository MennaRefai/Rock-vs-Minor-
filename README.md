# Rock-vs-Minor-
Built a minor classification model to predict rock frequency. Performed EDA, data preprocessing, and applied PCA for dimensionality reduction. Trained Random Forest Classifier and Logistic Regression, evaluating performance using accuracy and confusion matrix.
